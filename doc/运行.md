```
# 列出可用设备
flutter devices

flutter emulators

# 指定设备 ID 运行
flutter run -d <device-id>

# 例如指定 iOS 模拟器
flutter run -d "iPhone 15 Pro"

# 1. 查看有哪些模拟器（注意是 flutter emulators，不是 flutter run emulators）
flutter emulators

# 2. 启动指定模拟器
flutter emulators --launch Medium_Phone_API_36.0

# 3. 等模拟器启动完成后，再运行
flutter run -d emulator-5554
```

.vscode/launch.json
```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "ZheDuoDuo",
      "request": "launch",
      "type": "dart",
      "deviceId": "emulator-5554"
    }
  ]
}
```

```
# 运行到第一个可用的 Android 模拟器
flutter run -d $(flutter devices | grep emulator | head -1 | awk '{print $1}')
```
